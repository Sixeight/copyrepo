# copyrepo

`copyrepo` is a chrome extesion to copy GitHub style repository name.

## Install

1. Go chrome://extensions/
2. Enable `Developer Mode`
3. Import extension

## Usage

Try to right click at GitHub repository page, so you can find `Copy Repo` on context menu.
